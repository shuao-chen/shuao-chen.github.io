## 1. 引理陈述
强函数表示引理（Strong Functional Representation Lemma, SFRL）指出：
对于任意随机变量对 $(X,Y)\sim P_{XY}$，存在一个随机变量 $Z$，满足：
- $Z \perp X$；
- 存在确定性函数 $g$，使得
\begin{align}
Y = g(X,Z).
\end{align}
- 条件互信息满足
\begin{align}
I(X;Z|Y) \le \log\big(I(X;Y)+1\big) + 4.
\end{align}

## 2. 基数约束（离散情形）
若 $X,Y$ 为离散随机变量，且基数分别为 $\lvert \mathcal X \rvert,\lvert \mathcal Y \rvert$，则有：
\begin{align}
\lvert \mathcal Z \rvert \le \lvert \mathcal X \rvert (\lvert \mathcal Y \rvert - 1) + 2.
\end{align}

## 3. 等价形式推导
由互信息链式法则，
\begin{align}
I(X;Z|Y) + I(X;Y) = I(X;Y,Z).
\end{align}
另一方面，
\begin{align}
I(X;Y,Z) = I(X;Z) + I(X;Y|Z) = I(X;Y|Z),
\end{align}
其中使用了 $Z \perp X$。
又由于
\begin{align}
I(X;Y|Z) = H(Y|Z) - H(Y|X,Z),
\end{align}
而 $Y = g(X,Z)$，因此
\begin{align}
H(Y|X,Z)=0.
\end{align}
从而得到
\begin{align}
I(X;Y,Z)=H(Y|Z).
\end{align}
综上，
\begin{align}
H(Y|Z) \le I(X;Y) + \log\big(I(X;Y)+1\big) + 4.
\end{align}

## 4. 泊松函数表示构造（核心构造）
考虑一个带标记的泊松点过程：
\begin{align}
Z = \{(\tilde Y_i, T_i)\}_{i=1}^{\infty}.
\end{align}
其中：
- $\tilde Y_i \overset{\text{i.i.d.}}{\sim} P_Y$；
- 到达时间满足
\begin{align}
T_i - T_{i-1} \sim \mathrm{Exp}(1), \quad T_0=0.
\end{align}

### 4.1 索引选择
在给定 $x$ 和点集 $Z$ 时，定义：
\begin{align}
k(x,Z) = \arg\min_i \; t_i \cdot \frac{dP_Y}{dP_{Y\mid X}(\cdot\mid x)}(\tilde y_i).
\end{align}

### 4.2 构造函数
定义：
\begin{align}
g(x,Z) = \tilde y_{k(x,Z)}.
\end{align}
从而得到：
\begin{align}
Y = g(X,Z).
\end{align}

## 5. 直观理解
该构造的核心思想可以理解为：
- $Z$ 提供了一组“候选样本”（来自 $P_Y$）；
- $X$ 通过似然比选择最合适的样本；
- 从而模拟条件分布 $P_{Y\mid X}$。
本质上，这是用
> **独立随机性 + 确定性选择**
来实现原本的随机映射。

## 6. 个人理解
SFRL 的本质作用是：
> 将“随机映射 $P_{Y\mid X}$”转化为“确定函数 + 外部随机性”。

这一点在以下问题中非常关键：
- one-shot coding
- 信道模拟（channel simulation）
- 率失真问题构造
